## Configurar entorno
Para configurar el entorno virtual ejecutar los siguientes comandos dentro de la raíz del proyecto.
```sh
python3.12 -m venv .venv
source .venv/bin/activate
```

Una vez configurado el entorno virtual, instalar las dependencias del proyecto:
```sh
pip install -r requirements.txt
```

El último paso sería inciar Jupyter notebook a través del siguiente comando:
```sh
jupyter notebook
```
